# PharmaTrack Backend

PharmaTrack is a Pharma Management System backend built using Spring Boot.

## Tech Stack
- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- JWT Authentication

## Features
- User authentication & authorization
- Role-based access control
- Medicine & inventory management
- Prescription management
- Sales & billing

## Setup
1. Clone repository
2. Configure PostgreSQL in `application.properties`
3. Run the application

## API Documentation
Swagger UI: http://localhost:8080/swagger-ui.html
